# Smart Parking Application

## Tech Stack
- React
- Node.js (Express)
- MySQL (XAMPP)

## How to run
1. Start MySQL Workbench
2. Backend
   cd backend
   npm install
      npx sequelize-cli db:migrate
      npx sequelize-cli db:seed:all
   npm start
3. Frontend
   cd frontend
   npm install
   npm start
